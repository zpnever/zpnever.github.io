Penelitian & Rumus Ilmiah

Dalam penelitian sehari-hari, kami sangat bergantung pada pelarut universal seperti air murni yang memiliki rumus kimia H2O. Selain itu, untuk menganalisis data, terkadang kami menggunakan teori relativitas yang terkenal dengan persamaan E = mc2 atau perhitungan dasar luas lingkaran πr2.

Contoh Algoritma

Berikut adalah catatan dari algoritma sederhana yang sedang saya uji coba:

function hitungLuas(jariJari) {
    const PI = 3.14159;
    return PI * (jariJari * jariJari);
}

console.log("Luas:", hitungLuas(10));
